# morecats-mod
💥 模组理念

没有复杂合成链
没有工业流水线
没有多余系统

👉 这个模组只做一件事：战斗升级到极限

🔫 枪械系统
多种枪械（手枪 / 步枪 / 重型武器）
快节奏射击体验
简单获取，直接开打

👉 从冷兵器时代，直接跨进现代战争

☢️ 核武系统
小型核弹（战术清场）
大当量核弹（直接改地图）
远程发射 / 投掷机制

👉 一键改变地形，一次决定胜负

⚠️ 核心玩法
没有防护？你就会被秒
资源不重要，火力才重要
战斗 ≠ 对拼，而是碾压
🎮 游戏体验
⚡ 节奏极快（开局就能打）
💣 战斗极端（要么你炸别人，要么被炸）
🌍 世界不稳定（地图随时被抹平）
🧠 设计核心

👉 “最少内容，最大破坏力”
